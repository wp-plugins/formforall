== Description ==

FormForAll plugin allows you to insert the best forms directly coming from the FormForAll API.
The plugin is currently available in french.

== Installation ==

1. Upload the formforall directory to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Enter the API key in the Settings section.
4. Add a form when editing a post or add the widget to one of your sidebars.
